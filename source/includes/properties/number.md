# Number

**type** `number`

## Description

The number property is the HTML5 number input field. No custom validation exists only the browsers validation will kick in. The value is saved as a string and the output is a string.

## Settings

No settings exists.

## Example

```php
papi_property([
  'title' => 'Number',
  'slug'  => 'my_number_slug',
  'type'  => 'number'
])
```

## Example output

```php
int 10
```
