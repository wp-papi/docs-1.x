# String

**type** `string`

## Description

The string property is just a text input field. The value is saved as a string and the output is a string.

## Settings

No settings exists.

## Example

```php
papi_property([
  'title' => 'Name',
  'slug'  => 'my_name_slug',
  'type'  => 'string'
])
```

## Example output

```php
string 'Fredrik'
```
