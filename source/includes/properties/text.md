# Text

**type** `text`

## Description

This property will output the textarea tag and the output will be a string with all text from the textarea.

## Example

```php
papi_property([
  'title' => 'Text',
  'slug'  => 'my_text_slug',
  'type'  => 'text'
])
```

## Example output

```php
string 'Fredrik'
```
